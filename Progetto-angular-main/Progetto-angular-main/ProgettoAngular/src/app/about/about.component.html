CHI SIAMO?

Questo sito web è stato realizzato da Serhii Kushnir e Mustafa Vugdalic, studenti della classe 4^Ci dell'Istituto Tecnico Tecnologico "Marconi" di Rovereto.
Il nostro progetto si basa su un sito web che permette di visualizzare i dati relativi al Covid-19 in Italia. Nella pagina home è possibile visualizzare un grafico e diversi dati relativi alla situazione Covid-19 corrente e passata in Italia, mentre nella pagina overview è possibile visualizzare una mappa dell'Italia e cliccando su una regione è possibile visualizzare i dati attuali relativi a quella regione.
Il Progetto è stato realizzato in Angular, utilizzando il framework Tailwind CSS per la parte grafica, il framework Chart.js per la creazione del grafico e il database Firebase per il salvataggio dei dati.
